## 書籍管理サンプルアプリ

#### 環境

- Java 11

#### 実行方法

```bash
$ git clone https://github.com/miyazakijunichi/coding-exam-bookmanager.git
$ cd coding-exam-bookmanager
$ ./gradlew clean run
```

- 実行後、サーバーが立ち上がったら http://localhost:8080/ にアクセスできます
